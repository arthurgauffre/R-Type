Detailed Description

template<typename Clock, typename WaitTraits = asio::wait_traits<Clock>>
class asio::partial_cancel_at< Clock, WaitTraits >

A function object type that adapts a completion_token to cancel an operation at a given time. May also be used directly as a completion token, in which case it adapts the asynchronous operation's default completion token (or asio::deferred if no default is available).
