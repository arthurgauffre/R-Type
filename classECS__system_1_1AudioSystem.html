Updates the audio system.

Updates the audio system by processing sound and music components of entities.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to entities that are part of the system.
msgToSendA reference to a vector of pairs containing actions and their corresponding entity IDs to be sent.
msgReceivedA reference to a vector of pairs containing messages and their corresponding sender and receiver IDs.
entityMutexA reference to a mutex for synchronizing access to entities.
sceneStatusA shared pointer to the current scene status.

This function iterates through a list of entities and checks for the presence of SoundComponent and MusicComponent in each entity. If a SoundComponent or MusicComponent is found and is marked as playing, the respective play function is called.

deltaTimeThe time elapsed since the last update.
