Constructs a new SoundComponent object.

This constructor initializes a SoundComponent with the given entity ID and sound file path. It attempts to load the sound from the specified file path into a sound buffer. If the sound file cannot be loaded, it throws a runtime error.

Parameters
entityIDThe ID of the entity associated with this component.
soundFilePathThe file path to the sound file to be loaded.
Exceptions
std::runtime_errorIf the sound file cannot be loaded.
