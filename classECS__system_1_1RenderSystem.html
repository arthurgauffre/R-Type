Updates the render system.

Updates the render system by drawing all entities with the appropriate components.

This function updates the render system based on the elapsed time, the list of entities, messages to send, messages received, and the current scene status.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to entities that need to be updated.
msgToSendA reference to a vector of pairs containing actions and their corresponding entity IDs to be sent.
msgReceivedA reference to a vector of pairs containing messages and their corresponding entity IDs received.
entityMutexA reference to a mutex for synchronizing access to entities.
sceneStatusA shared pointer to the current scene status.

This function clears the window, processes entities with background and transform components to draw background sprites, and processes entities with transform, sprite, and texture components to draw textured sprites. Finally, it displays the window and handles window events.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to entities to be processed.

Updates the render system by drawing entities with specific components.

This function clears the window, locks the entity mutex, and iterates through entities with various component combinations to render them appropriately. It handles background sprites, regular sprites, rectangle shapes, text components, and filter rectangle shapes.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to entities to be processed.
msgToSendA reference to a vector of messages to be sent.
msgReceivedA reference to a vector of received messages.
entityMutexA mutex to ensure thread-safe access to entities.
sceneStatusA shared pointer to the current scene status.
