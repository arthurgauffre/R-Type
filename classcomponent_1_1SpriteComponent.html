Detailed Description

A component that represents a sprite in the game.

The SpriteComponent class is responsible for managing the position, color, and graphical representation of a sprite associated with an entity in the game. It inherits from the AComponent class.

Note
This class uses a shared pointer to an IGraphic interface for graphical operations.
