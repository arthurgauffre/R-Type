Detailed Description

template<typename Executor>
struct asio::partial_executor_binder< Executor >

A function object type that adapts a completion_token to specify that the completion handler should have the supplied executor as its associated executor. May also be used directly as a completion token, in which case it adapts the asynchronous operation's default completion token (or asio::deferred if no default is available).
