Updates the weapon system.

Updates the weapon system for all entities with a WeaponComponent.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to entities that are part of the system.
msgToSendA reference to a vector of pairs containing actions and their corresponding entity IDs to be sent.
msgReceivedA reference to a vector of pairs containing messages and their corresponding sender and receiver IDs.
entityMutexA reference to a mutex for synchronizing access to entities.
sceneStatusA shared pointer to the current scene status.

This function iterates through all entities that have a WeaponComponent and updates their state based on the current scene status, cooldowns, and firing status. It handles the creation of projectiles, cooldown management, and sound effects for firing weapons.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to all entities in the system.
msgToSendA reference to a vector of messages to be sent, containing actions and entity IDs.
msgReceivedA reference to a vector of received messages, containing message strings and entity ID pairs.
entityMutexA reference to a mutex for thread-safe entity access.
