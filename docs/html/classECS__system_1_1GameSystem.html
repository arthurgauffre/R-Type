Updates the game system.

Updates the game system by processing the entities.

This function updates the game system based on the elapsed time, the list of entities, messages to send, messages received, and the current scene status.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to the entities in the game.
msgToSendA reference to a vector of pairs containing actions and their corresponding entity IDs to be sent.
msgReceivedA reference to a vector of pairs containing messages and their corresponding sender and receiver IDs.
entityMutexA reference to a mutex for synchronizing access to the entities.
sceneStatusA shared pointer to the current scene status.

This function iterates through the provided entities and counts the number of players and enemies. If there are no players, it throws a GameLoosed exception indicating the game is over. If there are no enemies, it throws a GameWon exception indicating the player has won.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to the entities to be processed.
Exceptions
rtype::GameLoosedIf there are no players left in the game.
rtype::GameWonIf there are no enemies left in the game.
