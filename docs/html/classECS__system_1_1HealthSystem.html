Updates the health system for all entities.

Updates the health status of entities and deactivates those with zero or negative health.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to the entities to be updated.
msgToSendA reference to a vector of pairs containing actions and entity IDs to send messages.
msgReceivedA reference to a vector of pairs containing received messages with their corresponding entity IDs.
entityMutexA reference to a mutex for synchronizing access to entities.
sceneStatusA shared pointer to the current scene status.

This function iterates through a list of entities, checks their health status, and deactivates any entity whose health is less than or equal to zero.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to entities that need to be updated.
