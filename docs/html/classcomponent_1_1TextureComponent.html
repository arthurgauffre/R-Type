Updates the texture component.

Updates the texture component with a new texture.

This function is called to update the state of the texture component based on the elapsed time since the last update.

Parameters
deltaTimeThe time elapsed since the last update, in seconds.

This function updates the texture component by creating a new texture using the provided path and graphic interface. The texture ID is updated with the ID of the newly created texture.

Parameters
pathThe file path to the texture image.
graphicA shared pointer to the graphic interface used to create the texture.
