1 /*
2 ** EPITECH PROJECT, 2024
3 ** R-Type
4 ** File description:
5 ** Enum
6 **
7 ** This header file contains various enumerations used throughout the R-Type project.
8 ** Each enumeration represents a different category of constants that are used to
9 ** manage game scenes, actions, AI types, texture paths, text fonts, text strings,
10 ** sound paths, entity types, and keyboard inputs.
11 **
12 ** Enumerations:
13 ** - Scene: Represents different scenes in the game (e.g., MENU, GAME, GAMEOVER).
14 ** - Action: Represents different actions that can be performed (e.g., MOVE_UP, SHOOT, EXIT).
15 ** - AIType: Represents different types of AI behaviors (e.g., LINEAR, SINUSOIDAL).
16 ** - TexturePath: Represents different texture paths for game assets (e.g., Player, Enemy).
17 ** - TextFont: Represents different text fonts used in the game (e.g., Arial).
18 ** - TextString: Represents different text strings used in the game (e.g., Play, Menu).
19 ** - SoundPath: Represents different sound paths for game audio (e.g., Shoot, BackgroundMusic).
20 ** - Type: Represents different types of game entities (e.g., PLAYER, ENEMY, WEAPON).
21 ** - KeyBoard: Represents different keyboard inputs (e.g., A, B, C, Space, escape).
22 */
