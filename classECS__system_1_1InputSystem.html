Updates the input system.

Updates the input system for all entities with input components.

This function processes input actions and updates the state of entities accordingly.

Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to the entities to be updated.
msgToSendA reference to a vector of pairs containing actions and their corresponding entity IDs to be sent.
msgReceivedA reference to a vector of pairs containing messages and their corresponding sender and receiver IDs.
entityMutexA reference to a mutex for synchronizing access to entities.
sceneStatusA shared pointer to the current scene status.

This function processes input actions for entities and updates their velocity and firing state based on the input received. It iterates through all entities that have an InputComponent and performs the following actions:

  • Resets the entity's velocity to zero.
  • Updates the entity's velocity based on active input actions ("MoveUp", "MoveDown", "MoveLeft", "MoveRight").
  • Sets the entity's firing state if the "Shoot" action is active.
Parameters
deltaTimeThe time elapsed since the last update.
entitiesA vector of shared pointers to entities to be updated.
