Detailed Description

template<typename CompletionToken, typename Clock, typename WaitTraits = asio::wait_traits<Clock>>
class asio::cancel_at_t< CompletionToken, Clock, WaitTraits >

A completion_token adapter that cancels an operation at a given time.

The cancel_at_t class is used to indicate that an asynchronous operation should be cancelled if not complete at the specified absolute time.
